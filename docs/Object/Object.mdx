# Object

## Intro
Object is the base for all **drawable** objects. It is used as a base for anything that you want to draw

## Attributes
| Name     | Description |
| -------- | ----------- |
| name : string | The name of the object |
| zIndex : int | The layer the object is on |

## API Calls

### setName

```cpp
setName(name : string) -> void
```

Sets the name of the object.

Example: 
```lua
object:setName("newName")
```

### setZIndex

```cpp
setZIndex(zIndex : int) -> void
```

Sets the zIndex of the object, helps making layers with multiple objects in the same X and Y cords.

Example:
```lua
object:setZIndex(2)
```
