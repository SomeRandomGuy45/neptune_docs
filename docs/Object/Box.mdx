# Box

## Intro
Box is a simple square. The base of this class is [Object](/docs/Object/).

## Attributes
| Name     | Description |
| -------- | ----------- |
| x : float | Where the object is on the x line |
| y : float | Where the object is on the y line |
| Color : color | The color of the object |
